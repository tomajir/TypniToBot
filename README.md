# TipniToBot

Telegram bot pro prodej sportovních tiketů.

## Příkazy:
- /start – uvítání
- /dnes – dnešní tipy
- /zitra – zítřejší tipy
- /vip – přístup pouze pro VIP
- /kontakt – kontakt na admina

## Nasazení:
1. Nahraj na GitHub
2. Deploy na Railway
3. Přidej proměnnou `TOKEN` s tvým bot tokenem
